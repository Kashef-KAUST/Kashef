### Python Code Version 2: Interact with a Page Element

Building upon the initial code, we will now add functionality to click on a specific product. For demonstration purposes, I'll choose the "Sunglasses" product as an example. ```python
from selenium import webdriver
from selenium.webdriver.common.by import By
from selenium.webdriver.chrome.service import Service
from webdriver_manager.chrome import ChromeDriverManager
import time

# Initialize the WebDriver
driver = webdriver.Chrome(service=Service(ChromeDriverManager().install()))

# Navigate to the website
driver.get("https://cymbal-shops.retail.cymbal.dev")

# Wait for the page to load
time.sleep(5)

# Click on the "Sunglasses" product
sunglasses_link = driver.find_element(By.XPATH, "//a[@href='/product/OLJCESPC7Z']")

# Wait for the product detail page to load

# Print the HTML content of the product detail page
print(driver.page_source)

# Close the WebDriver
driver.quit()
