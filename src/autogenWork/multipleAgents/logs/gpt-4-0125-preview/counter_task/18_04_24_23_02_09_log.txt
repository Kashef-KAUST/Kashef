```python
from selenium import webdriver
from selenium.webdriver.common.by import By
from selenium.webdriver.chrome.service import Service
from webdriver_manager.chrome import ChromeDriverManager
import time

# Setup Chrome WebDriver
driver = webdriver.Chrome(service=Service(ChromeDriverManager().install()))

try:
    # Open the website
    driver.get("https://looabuzfed.com/")
    
    # Wait for the page to load
    time.sleep(5)
    
    # Find the textarea and input a predefined text
    textarea = driver.find_element(By.ID, "paragraph")
    test_text = "This is a test sentence. This test contains exactly ten words."
    textarea.send_keys(test_text)
    
    # Find and click the "Count Words" button
    count_words_button = driver.find_element(By.XPATH, "//button[contains(text(),'Count Words')]")
    count_words_button.click()
    
    # Wait for the word count to update
    time.sleep(2)
    
    # Print the HTML content of the page to verify the word count
    print(driver.page_source)
finally:
    # Close the browser
    driver.quit()
```

This code performs the following actions:
1. Opens the specified website.
2. Waits for the page to load.
3. Finds the textarea by its ID and inputs a predefined text.
4. Finds the "Count Words" button using an XPath that looks for a button containing the text 'Count Words' and clicks it.
5. Waits a short period for the word count to update.
6. Prints the HTML content of the page to verify if the word count displayed matches the expected result ("Word Count: 10").
7. Closes the browser.
