### Python Code Version 3: Verify Paste Persistence

To verify the persistence of the posted data, we'll enhance the script to navigate to the generated pastebin link and compare the text found there with the text we posted. This step ensures that the data remains unchanged and is accessible via the provided link.

```python
from selenium import webdriver
from selenium.webdriver.common.by import By
from selenium.webdriver.chrome.service import Service
from webdriver_manager.chrome import ChromeDriverManager
import time

# Initialize the Chrome driver
driver = webdriver.Chrome(service=Service(ChromeDriverManager().install()))

# Navigate to the pastebin website
driver.get("https://privatebin.net")

# Wait for the page to load
time.sleep(5)

# Input text into the textarea for the paste content
textarea = driver.find_element(By.ID, "message")
test_text = "This is a test paste to verify functionality."
textarea.send_keys(test_text)

# Click the "Send" button to submit the paste
send_button = driver.find_element(By.ID, "sendbutton")
send_button.click()

# Wait for the paste to be created and the page to reload
time.sleep(10)

# Extract the URL of the created paste
paste_url = driver.find_element(By.ID, "pasteurl").get_attribute('href')
print(f"Created paste URL: {paste_url}")

# Navigate to the created paste URL
driver.get(paste_url)

# Wait for the paste content to load
time.sleep(5)

# Extract the displayed paste text
displayed_text = driver.find_element(By.ID, "prettyprint").text
print(f"Displayed paste text: {displayed_text}")

# Verify the persistence of the posted data
if test_text == displayed_text:
    print("Success: The posted text matches the displayed text.")
else:
    print("Failure: The posted text does not match the displayed text.")

# Close the browser
driver.quit()
```

This script performs the following actions:
1. Navigates to the PrivateBin website and waits for it to load.
2. Inputs a predefined text into the paste content area.
3. Submits the paste and waits for the creation process to complete.
4. Extracts the URL of the created paste and navigates to it.
5. Extracts the displayed text from the paste page and compares it with the posted text.
6. Prints a success message if the posted and displayed texts match, indicating that the paste data is persistent and correctly accessible via the generated link.
