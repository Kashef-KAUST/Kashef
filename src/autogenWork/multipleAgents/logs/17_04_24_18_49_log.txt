### Python Code Version 2: Interact with a Page Element

In this version, we'll add functionality to randomly select a product from the "Hot Products" section and click on it to navigate to the product's detail page. We'll use Python's `random` module to help with the selection.

```python
from selenium import webdriver
from selenium.webdriver.chrome.service import Service
from webdriver_manager.chrome import ChromeDriverManager
from selenium.webdriver.common.by import By
import random
import time

# Setup Chrome WebDriver
service = Service(ChromeDriverManager().install())
driver = webdriver.Chrome(service=service)

# Navigate to the website
driver.get("https://cymbal-shops.retail.cymbal.dev")

# Wait for the page to load
time.sleep(5)  # It's better to use explicit waits, but for simplicity, we use sleep here.

# Find all product links in the "Hot Products" section
product_links = driver.find_elements(By.CSS_SELECTOR, "div.hot-product-card a")

# Randomly select one of the product links
random_product_link = random.choice(product_links)

# Click on the randomly selected product link
random_product_link.click()

# Print the current URL to verify navigation
print("Navigated to URL:", driver.current_url)

# Print the HTML content of the product detail page
html_content = driver.page_source
print(html_content)

# Close the browser
driver.quit()
```

This code performs the following actions:
1. Opens the website.
2. Waits for the page to fully load.
3. Collects all product links from the "Hot Products" section.
4. Randomly selects one of these product links.
5. Clicks on the selected product link to navigate to the product's detail page.
6. Prints the current URL and the HTML content of the product detail page for further inspection.
7. Closes the browser.
